# ЛУННЫЙ АТЛАС

## СОДЕРЖАНИЕ

Лунный атлас — хранилище для [Obsidian](https://obsidian.md/), предназначенное для хранения материалов в жанре [Worldbuilding](https://en.wikipedia.org/wiki/Worldbuilding).

## ИСПОЛЬЗОВАНИЕ

1. Скачайте и установите приложение Obsidian с [официального сайта](https://obsidian.md).
2. Загрузите данный репозиторий, выбрав опцию `"Code" → "Download ZIP"`.
3. В главном меню Obsidian выберите пункт `"Открыть папку как хранилище"` и укажите директорию `Lunar Atlas`.
