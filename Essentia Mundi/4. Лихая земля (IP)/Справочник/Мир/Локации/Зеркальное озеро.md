# ЗЕРКАЛЬНОЕ ОЗЕРО 

## 🟄

Зеркальное озеро ([[Программа глобального исследования|GPR-LCT-002]]) — тип водоёма аномального характера, ассоциируемый с [[Повелительница озёр|Повелительницей озёр]].

Отличается зеркально гладкой поверхностью и почти полным отсутствием сезонного ледостава.

Среднегодовая температура воды не превышает 0 °C.
